# Иконки для нижнего меню

Пожалуйста, поместите следующие изображения в эту папку:

1. `inventory.png` - Иконка инвентаря (3D коробка)
2. `boxes.png` - Иконка коробок (3D куб)
3. `scanner.png` - Иконка сканера (камера)
4. `delivery.png` - Иконка партий (грузовик)
5. `unpacking.png` - Иконка распаковки (переплетенные коробки)
6. `tasks.png` - Иконка задач (документ с карандашом)
7. `profile.png` - Иконка профиля (силуэт человека)

Все изображения должны быть в формате PNG с прозрачным фоном.
Рекомендуемый размер: 60x60 пикселей или больше (будут автоматически масштабироваться).
