# Как сохранить проект как архив

## Способ 1: Через проводник Windows (самый простой)

1. Откройте папку `Desktop` (Рабочий стол)
2. Найдите папку `packgo3`
3. **Правой кнопкой мыши** на папке `packgo3`
4. Выберите **"Отправить"** → **"Сжатая ZIP-папка"**
5. Архив `packgo3.zip` будет создан в той же папке

## Способ 2: Через командную строку

Откройте PowerShell или командную строку в папке Desktop и выполните:

```powershell
Compress-Archive -Path packgo3 -DestinationPath packgo3.zip
```

## Способ 3: Использовать готовый скрипт

Дважды кликните на файл `create-archive.bat` в папке проекта - он создаст архив автоматически.

## Способ 4: Через 7-Zip или WinRAR

1. Установите 7-Zip или WinRAR (если еще не установлены)
2. Правой кнопкой мыши на папке `packgo3`
3. Выберите **"7-Zip"** → **"Добавить в архив..."** или **"WinRAR"** → **"Добавить в архив..."**
4. Выберите формат (ZIP, RAR, 7z) и нажмите OK

## Что включено в архив

- ✅ Весь код frontend (HTML, CSS, JavaScript)
- ✅ Весь код backend (Node.js сервер)
- ✅ Файлы конфигурации (package.json)
- ✅ Документация (README.md)

## Что НЕ включено (и не нужно включать)

- ❌ `node_modules/` - папка с зависимостями (создается через `npm install`)
- ❌ `database.sqlite` - база данных (создается автоматически при первом запуске)
- ❌ `.git/` - папка Git (если используете Git)

## После создания архива

Архив можно:
- Отправить по email
- Загрузить в облачное хранилище (Google Drive, Dropbox, etc.)
- Сохранить на флешку или внешний диск
- Загрузить на GitHub или другой Git-хостинг

## Восстановление проекта из архива

1. Распакуйте архив `packgo3.zip`
2. Перейдите в папку `backend`
3. Выполните `npm install` для установки зависимостей
4. Запустите сервер командой `npm start`
